### 7. Кнопки — мобильные и десктоп

**Файлы:** `css/style.css`, `js/main.js`

#### Десктоп (≥769px) — боковые плавающие кнопки
```css
.float-btns {
  position: fixed; right: 0; top: 50%;
  transform: translateY(-50%); z-index: 900;
}
.float-btn { width: 200px; height: 52px; /* без анимации */ }
```

4 кнопки:
| Кнопка | Действие |
|--------|----------|
| Написать нам | `mailto:info@airbond.ru` |
| Рассчитать систему | `#calculator` (скролл) |
| Спросить Алекса | открывает AI-чат пузырь |
| Telegram | `https://t.me/airbond_barnaul_bot` |

#### Мобильный (≤768px) — фиксированная панель снизу
```css
@media (max-width: 768px) {
  .float-btns { display: none; }  /* боковые скрыты */
  .mobile-bar {
    display: flex; position: fixed;
    bottom: 0; left: 0; right: 0;
    padding: 10px 12px calc(10px + env(safe-area-inset-bottom));
  }
}
```

2 кнопки:
| Кнопка | Действие |
|--------|----------|
| Telegram (синяя) | `https://t.me/airbond_barnaul_bot` |
| Получить расчёт (зелёная) | `openLeadModal('float')` |

**Причина замены:** на Android Chrome любое `%` или `vh` в `position: fixed` пересчитывается когда адресная строка прячется/появляется → кнопки прыгали. `bottom: 0` на фиксированной панели этой проблемы не имеет.
